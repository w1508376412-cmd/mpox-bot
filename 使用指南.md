## 🔧 常见问题

### Q1: 启动失败，提示端口被占用？
**A:** 停止占用端口的进程：
```bash
kill $(lsof -ti:8000)
```

### Q2: 找不到数据？
**A:** 重新添加示例数据：
```bash
cd "/Users/yanfei/Library/Application Support/0011/workspaces/default/mpox-bot"
source venv/bin/activate
python backend/add_sample_data.py
```

### Q3: API调用失败？
**A:** 检查DeepSeek API Key是否有效：
```bash
cat .env | grep OPENAI_API_KEY
```

### Q4: 如何添加更多知识？
**A:** 编辑 `backend/add_sample_data.py`，添加更多示例数据，然后运行：
```bash
python backend/add_sample_data.py
```
